## 02 · OPE Carranque

Este chat es solo para Carranque. Estoy apuntado. Usa las bases de Carranque: 1 plaza + bolsa, test de 80 preguntas y práctico sobre temas 3 a 20. Calcula mis puntos con DAM como FP Grado Superior si fue acreditado, y sé conservador con formación/experiencia si no consta aportada en plazo.

## 03 · OpoWeb modificaciones

Este chat es solo para modificar OpoWeb. Quiero una web estática para GitHub Pages, usable en móvil/tablet, con selector de oposición, biblioteca normativa reutilizable, tests corregibles con justificación, supuestos, simulacros, comparador de baremos, calendario y control de cambios. No metas datos personales en el repositorio.

## 04 · Formación y méritos globales

Este chat es para subir mis certificados, títulos, cursos, vida laboral y méritos. Extrae nombre del curso, entidad, fecha, horas, materia y posible encaje por convocatoria. Devuélveme una tabla: válido/no válido/dudoso para La Puebla, Carranque, Diputación y UC3M. No sumes puntos si el mérito no fue aportado en plazo o no cumple las bases. En Diputación Administrativo C1 no sumes méritos porque es oposición libre. En UC3M solo valores inglés oficial admitido, experiencia como funcionario en escala equivalente y cursos válidos según bases.
