# Estructura de chats para el proyecto

## 01 · OPE La Puebla de Montalbán

Este chat es solo para La Puebla de Montalbán. Usa mi perfil base: DAM, sin experiencia administrativa pública confirmada salvo que yo indique lo contrario. Quiero análisis realista de bases, listados, plazos, baremo, test, supuestos y bolsa. No mezcles esta convocatoria con Carranque ni Las Ventas salvo que te pida comparar.

## 02 · OPE Carranque

Este chat es solo para Carranque. Estoy apuntado. Usa las bases de Carranque: 1 plaza + bolsa, test de 80 preguntas y práctico sobre temas 3 a 20. Calcula mis puntos con DAM como FP Grado Superior si fue acreditado, y sé conservador con formación/experiencia si no consta aportada en plazo.

## 03 · OPE Las Ventas con Peña Aguilera

Este chat es solo para Las Ventas con Peña Aguilera. Trabaja con la convocatoria de 1 plaza de Auxiliar Administrativo, concurso-oposición libre. Revisa plazos, solicitud, méritos, test, supuestos prácticos y estrategia. Usa mi perfil base y no des por válidos cursos si no cumplen exactamente las bases.

## 04 · OpoWeb modificaciones

Este chat es solo para modificar OpoWeb. Quiero una web estática para GitHub Pages, usable en móvil/tablet, con selector de oposición, temario común, tests corregibles con justificación, supuestos, simulacros, comparador de baremos, calendario y control de cambios. No metas datos personales en el repositorio.

## 05 · Formación y méritos globales

Este chat es para subir mis certificados, títulos, cursos, vida laboral y méritos. Extrae nombre del curso, entidad, fecha, horas, materia y posible encaje por convocatoria. Devuélveme una tabla: válido/no válido/dudoso para La Puebla, Carranque y Las Ventas. No sumes puntos si el mérito no fue aportado en plazo o no cumple las bases.
