# Estándar de temario autosuficiente de OpoWeb

Un tema autosuficiente permite estudiar el epígrafe oficial sin manual externo.

## Requisitos

Título literal, resumen, fuentes oficiales y fecha, cobertura normativa o funcional, desarrollo completo, reglas y excepciones, aplicación, síntesis, puntos calientes, tres preguntas de retención, estrategia, esquema, tabla y banco coherente.

## Barreras automáticas

- Carranque: mínimo 1.000 palabras.
- La Puebla: mínimo 900 palabras.
- Dos fuentes oficiales.
- Estructura fija y título coincidente.
- Tabla, esquema y treinta preguntas.
- Ninguna sección genérica o pendiente.

Los mínimos no sustituyen revisión jurídica o técnica.

## Fuentes

Materias jurídicas: BOE, DOUE, diarios y sedes oficiales. Informática: documentación oficial de Microsoft, LibreOffice, estándares y fabricantes. No se usan blogs, academias o resúmenes comerciales como autoridad.
